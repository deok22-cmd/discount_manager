## 2. 주요 기능 상세

### 📊 대시보드 (Dashboard)
*   **실시간 통계**: 전체 발행권 수, 사용 완료 건수, 사용 가능(대기) 건수를 요약하여 표시합니다.
*   **최근 발행 내역**: 가장 최근에 발행된 10건의 데이터를 리스트로 보여줍니다. (고객명, 이벤트명, 쿠폰번호, 상태, 발행일, **만료일** 포함)

### 📅 이벤트 관리 (Event Management)
*   **이벤트 등록/수정**:
    -   **이벤트명**: 행사의 고유 명칭.
    -   **할인 대상**: 드롭다운 선택 (텐트세탁비, 우레탄창, 발수코팅, 곰팡이제거, 배송비, 기타).
    -   **할인 설정**: 고정 금액(원) 또는 퍼센트(%) 선택 및 수치 입력.
    -   **유효 기간**: 발행일로부터 고객이 사용할 수 있는 기간(일수)을 숫자로 입력 (예: 50일).
    -   **상세 메모**: 관리자용 내부 메모 필드.
    -   **커스텀 메시지 템플릿**: 고객에게 발송될 메시지 내용을 이벤트별로 설정. 치환자(`[이벤트명]`, `[할인대상]`, `[할인수치]`, `[할인유형]`, `[유효기간]`, `[쿠폰번호]`)를 활용한 동적 메시지 생성 지원.
*   **쿠폰 발행 현황 조회**: 이벤트별로 발행된 전체 리스트를 **전체 화면**에서 조회하고, 고객 성함이나 연락처로 실시간 검색할 수 있습니다.

### 🎟️ 할인권 개별 발행하기 (Issue Coupon)
*   **발행 프로세스**:
    1.  이벤트 선택 (형식: `이벤트명 (할인대상 수치유형)`)
    2.  고객 성함 및 연락처 입력
    3.  **날짜 자동 계산**: 발행 시점 기준 유효 기간을 계산하여 만료 날짜 확정.
    4.  **메시지 생성**: 설정된 템플릿에 따라 완성된 안내 문구 출력.
    5.  **복사 기능**: [문구 복사하기] 버튼으로 즉시 카카오톡이나 SMS에 붙여넣기 가능.

### 🔍 쿠폰 조회 및 사용 처리 (Verification)
*   **실시간 검증**: 쿠폰 번호(예: TK-XXXXXX)를 입력하여 유효성, 유효기간, 사용 여부를 즉시 확인합니다.
*   **사용 완료 처리**: 조회된 쿠폰이 '발행완료' 상태일 때, [할인권 사용 처리하기] 버튼을 통해 '사용완료'로 상태를 변경하여 중복 사용을 방지합니다.

## 3. 데이터베이스 구조 (DB Schema)

### `events` 테이블
| 컬럼 | 타입 | 설명 |
| :--- | :--- | :--- |
| `id` | INT (PK) | 고유 번호 |
| `event_name` | VARCHAR | 이벤트 명칭 |
| `discount_type` | ENUM | FIXED(원) / PERCENT(%) |
| `discount_target`| VARCHAR | 할인 대상 항목 |
| `discount_value` | INT | 할인 금액 또는 비율 |
| `valid_days` | INT | 유효 기간(일수) |
| `description` | TEXT | 상세 메모 |
| `msg_template` | TEXT | 고객 안내 문구 템플릿 |
| `created_at` | DATETIME | 생성 일시 |

### `coupons` 테이블
| 컬럼 | 타입 | 설명 |
| :--- | :--- | :--- |
| `id` | INT (PK) | 고유 번호 |
| `event_id` | INT (FK) | 연결된 이벤트 ID |
| `coupon_code` | VARCHAR | 랜덤 생성된 쿠폰 번호 (예: AB123456) |
| `customer_name` | VARCHAR | 고객 성함 |
| `phone_number` | VARCHAR | 고객 연락처 |
| `status` | ENUM | ISSUED(발행) / USED(사용완료) |
| `expiration_date`| DATE | 계산된 특정 만료 날짜 |
| `issued_at` | DATETIME | 발행 일시 |

## 4. 기술 스택 및 디자인
*   **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
    -   **Lucide Icons**: 세련된 아이콘 적용.
    -   **Glassmorphism**: 반투명 카드와 부드러운 그린 테마 UI.
    -   **SPA 방식**: 페이지 새로고침 없이 탭과 뷰 전환.
*   **Backend**: PHP 7/8 (PDO 방식의 안전한 DB 통신)
*   **Security**: 세션 기반 로그인 처리, SQL Injection 방지 처리가 되어 있으며, `.gitignore`를 통해 운영 핵심 정보(SSH Key 등) 노출 차단.
